How to Add a Wishlist Text

1. Download the text from the source in your browser
2. Save as .txt in 02_raw_sources/Library_/SacredTexts.com/ (or appropriate dir)
3. Tell Claude: "ingest [filename]"
4. Pipeline runs automatically:
   python 05_scripts/tag_frontmatter.py --write
   python 05_scripts/export_reader_data.py
   python 05_scripts/gzip_web_data.py
   cd 03_web_app && git add ... && git push
